DRIVER DROWSINESS DETECTION SYSTEM

Developed a drowsiness Detection system by leveraging deep learning and machine learning algorithms to detect signs of driver fatigue in real-time. Our system captures video from the driver’s face and analyzes it for drowsiness signs such as yawning, blinking, and head movement. If the system detects fatigue, it triggers an alert to warn the driver, reducing the risk of accidents.

See More

Key Features

  • Alert System
  • Facial Landmark Detection
  • CNN and pre-defined VGG-16 utilization
  • Maximum Acuuuracy within less number of epoches

Technologies Used

  • Open CV
  • Python
  • Data Augumentation Techniques

Challenges Faced and Solutions

  • Face Detection in Different Lighting: Ensuring accurate detection under various lighting conditions was challenging. I used **OpenCV’s Histogram Equalization** technique to enhance face detection in low light.
  • Model Training: Training the machine learning model with limited data was a challenge, but I overcame it by augmenting the dataset with synthetic images of drowsy faces.

Future Enhancements:

  • Real-Time Driver Monitoring
  • Emotion detection
  • Controlling vehicle steering based on drowsiness level
